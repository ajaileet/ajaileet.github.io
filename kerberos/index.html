Attack Cards
KERBEROASTING
MEDIUM
Request TGS tickets for accounts with SPNs. Tickets are encrypted with the service account's NTLM hash — crack offline.
Requires
Any valid domain user
Result
Service account plaintext password
GetUserSPNs.py Rubeus kerberoast Hashcat -m 13100
AS-REP ROASTING
MEDIUM
Target accounts with pre-authentication disabled. AS-REP responses are encrypted with user's hash — no auth needed.
Requires
Preauth disabled on account
Result
User plaintext password
GetNPUsers.py Rubeus asreproast Hashcat -m 18200
GOLDEN TICKET
CRITICAL
Forge a TGT using the krbtgt hash. Impersonate any user in the domain with any group membership. Survives password resets.
Requires
krbtgt NTLM hash (DA)
Result
Persistent domain admin access
mimikatz impacket-ticketer Rubeus
SILVER TICKET
HIGH
Forge a TGS for a specific service using the service account hash. No KDC contact — harder to detect. CIFS, HTTP, MSSQL, HOST.
Requires
Service account NTLM hash
Result
Access to specific service as any user
impacket-ticketer mimikatz klist
DIAMOND TICKET
CRITICAL
Request a real TGT, decrypt & modify PAC (add groups), re-sign. Bypasses Golden Ticket detections because KDC signature is valid.
Requires
krbtgt hash or AES key
Result
Stealthy domain-wide persistence
Rubeus diamond
SAPPHIRE TICKET
STEALTH
Clone a real user's PAC from LDAP into a forged ticket. PAC matches actual AD data — evades anomaly-based detection.
Requires
krbtgt hash + LDAP read access
Result
Most evasive forged ticket variant
Rubeus
PASS-THE-TICKET
HIGH
Inject a stolen or forged Kerberos ticket (.ccache / .kirbi) into the current session to authenticate as another user or service.
Requires
Valid .ccache or .kirbi ticket
Result
Lateral movement without plaintext creds
mimikatz KRB5CCNAME=ticket.ccache Rubeus ptt
OVERPASS-THE-HASH
HIGH
Use a user's NTLM hash or AES key to request a legitimate TGT from the KDC. Elevates NTLM hash to full Kerberos ticket.
Requires
NTLM hash or AES key
Result
Full TGT → lateral movement
mimikatz sekurlsa::pth getTGT.py Rubeus asktgt
DCSYNC
CRITICAL
Impersonate a domain controller using MS-DRSR to replicate credentials. Dumps all hashes including krbtgt without touching lsass.
Requires
Replicating Directory Changes + All
Result
All domain hashes → full compromise
secretsdump.py mimikatz lsadump::dcsync
UNCONSTRAINED DELEGATION
CRITICAL
Any user's TGT is stored in memory of a server with unconstrained delegation. Combine with Printer Bug to coerce DC authentication.
Requires
Compromise server with TrustedForDelegation
Result
Steal DC TGT → Domain Admin
Rubeus monitor SpoolSample mimikatz
CONSTRAINED DELEGATION
HIGH
Abuse S4U2Self + S4U2Proxy to impersonate any user (including DA) to allowed target services — no password needed.
Requires
Service account with delegation configured
Result
Admin impersonation on target services
getST.py -impersonate Rubeus s4u
RBCD ABUSE
CRITICAL
Write msDS-AllowedToActOnBehalfOfOtherIdentity on a computer object. Create fake machine account, abuse S4U to impersonate DA.
Requires
Write on target computer object
Local admin on target as DA
Result
PowerView addcomputer.py getST.py
SKELETON KEY
CRITICAL
Patch lsass on a DC to accept a master password for all accounts. Existing passwords still work — stealth backdoor.
Requires
Domain Admin + DC access
Result
Persistent in-memory backdoor
mimikatz misc::skeleton
